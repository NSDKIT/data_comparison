# データ比較ツール

EP / EV の Excel データを取り込んで比較分析するWebアプリです。

## ローカルでの起動方法

**必要なもの:** Node.js 18 以上

```bash
cd app
npm install
npm run dev
```

ブラウザで http://localhost:5173 を開いてください。

## 使い方

1. **EP データ**（`.xlsx` / `.xls`）をドロップまたはクリックして選択
2. **EV データ**（`.xlsx` / `.xls`）をドロップまたはクリックして選択
3. 両ファイルの読み込みが完了したら「比較分析を開始」ボタンが有効になります
